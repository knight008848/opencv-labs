## 概念 A：图像算术与混合

### 一句话解释
两张图可以像数字一样加减乘除——叠加水印、计算差异、做淡入淡出效果。

### 生活类比
图像加法 = 把两张半透明的幻灯片叠在一起投影，两层的颜色叠加后变亮。图像减法 = 拍了两张教室照片（一张有人一张没人），相减后剩下的就是"多出来的那个人的轮廓"。`addWeighted` = 调音台上推两个轨道的音量滑块，控制 A 和 B 各占多少比例。

### 技术解释
`cv2.add(img1, img2)` 是饱和加法——超过 255 的值截断为 255（不会溢出回绕）。NumPy 的 `img1 + img2` 是模运算——256 变成 0（溢出）。`cv2.addWeighted(img1, α, img2, β, γ)` 计算 `α×img1 + β×img2 + γ`，α+β 通常等于 1。

```python
blended = cv2.addWeighted(img1, 0.7, img2, 0.3, 0)  # 70% img1 + 30% img2
diff = cv2.absdiff(img1, img2)                        # 差异图
```

### 真实案例
手机相机的 HDR 模式：拍三张不同曝光的照片（欠曝、正常、过曝），用加权混合合成一张——亮处取欠曝的、暗处取过曝的，最终照片亮部和暗部都有细节。

## 概念 B：位运算与掩膜

### 一句话解释
位运算（AND/OR/NOT/XOR）按像素的每一个二进制位做逻辑操作，配合掩膜可以实现"只修改图片的某些区域"。

### 生活类比
你在白纸上盖一张镂空的纸板（掩膜），然后喷漆——只有镂空处被喷到颜色。AND 操作 = 只有掩膜是白的区域才保留原图颜色。OR 操作 = 把两层的颜色叠加。NOT = 照片变底片（黑白反转）。

### 技术解释
`cv2.bitwise_and(img, img, mask=mask)` 是最常用的组合——掩膜中白色(255)的区域保留原图像素，黑色(0)的区域变黑。这比 ROI 更灵活，因为掩膜可以是任意形状的。

```python
mask = cv2.inRange(hsv, lower, upper)          # 颜色掩膜
result = cv2.bitwise_and(img, img, mask=mask)  # 只保留掩膜区域
```

### 真实案例
绿幕抠像：在 HSV 空间中找到绿色的范围，生成掩膜（绿色=255，其余=0），然后 `bitwise_and` 把绿色区域替换成想要的背景。

## 概念 C：HSV 色彩空间与 `inRange`

### 一句话解释
HSV 把颜色拆成三个独立维度——什么颜色（H）、多鲜艳（S）、多亮（V），让你可以不管光照变化，只按"色相"找红色物体。

### 生活类比
BGR 像调油漆——"加 30% 蓝、50% 绿、20% 红"。HSV 像描述一个颜色——"大红（色相 H），很鲜艳（饱和度 S），比较亮（明度 V）"。车间里光照一会亮一会暗，用 BGR 找红色工件（R>200, G<50, B<50）今天调好明天就失效。用 HSV 的 H 通道（红色始终在 0-10 和 160-179），光照变化只影响 V 不影响 H。

### 技术解释
OpenCV 中 HSV 的范围：H=[0,179]（不是标准的 0-360，被压缩了一半），S=[0,255]，V=[0,255]。`cv2.inRange(hsv, lower, upper)` 返回二值掩膜——在范围内的像素=255，范围外的=0。红色的 H 跨了两个区间（0-10 和 160-179），因为色相是环状的。

```python
hsv = cv2.cvtColor(img, cv2.COLOR_BGR2HSV)
lower_red = np.array([0, 100, 100])
upper_red = np.array([10, 255, 255])
mask = cv2.inRange(hsv, lower_red, upper_red)
```

### 真实案例
水果分拣线：传送带上苹果（红）、橙子（橙）、青柠（绿）混在一起。三个 `inRange` 分别提取三种颜色的掩膜，各自计算质心位置，机械臂根据颜色分类抓取。

## 参考答案

Q1: C — OpenCV 的 `add` 做饱和运算，超过 255 截断为 255

Q2: Hue（色相）、Saturation（饱和度）、Value（明度）；H（色相）

Q3: B — 色相是环状的，红色跨了 0° 附近的区间

Q4: 错误 — HSV 空间将颜色和亮度解耦，做颜色过滤远优于 BGR

Q5: 深红和浅红的 H（色相）相同，S（饱和度）也相似，区别主要在 V（明度）。如果深浅都要，就放宽 V 的范围（如 50-255）。
