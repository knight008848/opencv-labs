## 概念 A：三种二值化策略

### 一句话解释
二值化 = 将灰度图变成纯黑(0)纯白(255)——设定一个阈值，亮的变白、暗的变黑。

### 生活类比
老师在黑板上用白粉笔写字。你想只保留粉笔字——"比这个灰度亮的都是粉笔（白），暗的都是黑板（黑）"。但如果教室左边有窗（亮）、右边没窗（暗），全局阈值就失败了——左边黑板看起来都很亮。自适应阈值 = 把黑板分成小格子，每个格子自己判断"什么算亮"。

### 技术解释
- **全局阈值** `cv2.threshold(gray, 127, 255, THRESH_BINARY)`：全图统一阈值，光照均匀时好用
- **Otsu** `THRESH_BINARY + THRESH_OTSU`：算法自动找最佳分割点（使类间方差最大），适合双峰直方图
- **自适应** `cv2.adaptiveThreshold(gray, 255, ADAPTIVE_THRESH_GAUSSIAN_C, THRESH_BINARY, 31, 10)`：每个 31×31 的小区域自己算阈值。参数 10 是从均值中减去的常数（越大=白色越少）

```python
_, global_t = cv2.threshold(gray, 127, 255, cv2.THRESH_BINARY)
_, otsu = cv2.threshold(gray, 0, 255, cv2.THRESH_BINARY + cv2.THRESH_OTSU)
adaptive = cv2.adaptiveThreshold(gray, 255,
    cv2.ADAPTIVE_THRESH_GAUSSIAN_C, cv2.THRESH_BINARY, 31, 10)
```

### 真实案例
试卷扫描阅卷：拍照上传后，自适应二值化处理——即使学生用不同颜色的笔、写在光照不均的答题卡上，最终都能变成清晰的黑白文字供 OCR 识别。

## 概念 B：腐蚀与膨胀

### 一句话解释
腐蚀 = 白色区域"瘦一圈"，膨胀 = 白色区域"胖一圈"——用一个小形状（结构元素）在二值图上滑动，决定每个像素的生死。

### 生活类比
腐蚀 = 沙滩上的沙堡被海浪冲——外围的沙子被冲走，沙堡变小。膨胀 = 往面包上涂黄油——面包（白色）变厚一圈，两片面包之间的小缝隙可能被黄油填满连在一起。

### 技术解释
腐蚀：只有当结构元素覆盖的所有像素都是 255 时，中心像素才保持 255。膨胀：只要结构元素覆盖的区域有任意一个 255，中心像素就变 255。`iterations` 参数控制重复次数。

```python
kernel = cv2.getStructuringElement(cv2.MORPH_RECT, (5, 5))
eroded = cv2.erode(binary, kernel, iterations=1)
dilated = cv2.dilate(binary, kernel, iterations=1)
```

### 真实案例
细胞计数：显微镜下的细胞图经过二值化后，相邻细胞可能粘连在一起。用腐蚀操作将粘连的细胞分开，再数轮廓数量。腐蚀的迭代次数 = 粘连程度 / 结构元素大小。

## 概念 C：开运算与闭运算

### 一句话解释
开运算 = 先腐蚀再膨胀 → 去小白噪点 + 断开细连接。闭运算 = 先膨胀再腐蚀 → 填小黑洞 + 连接断线。

### 生活类比
开运算 = 先拿砂纸把毛刺磨掉（腐蚀），再涂一层保护漆（膨胀）——毛刺没了，大块结构还在。闭运算 = 先把裂缝用腻子填平（膨胀），再把多余的腻子打磨掉（腐蚀）——裂缝没了，整体尺寸不变。

### 技术解释
`cv2.morphologyEx(binary, cv2.MORPH_OPEN, kernel)` 等价于 `dilate(erode(binary))`。`MORPH_CLOSE` 等价于 `erode(dilate(binary))`。还有 `MORPH_GRADIENT`（膨胀-腐蚀=物体外轮廓带）和 `MORPH_TOPHAT`（原图-开运算=亮噪点）等。

```python
opening = cv2.morphologyEx(binary, cv2.MORPH_OPEN, kernel)
closing = cv2.morphologyEx(binary, cv2.MORPH_CLOSE, kernel)
```

### 真实案例
指纹识别预处理：原始指纹图像有很多细小的断纹和噪点。闭运算连接断开的纹线，开运算去除纹线间的细小噪点。处理后指纹纹线连续、背景干净。

## 参考答案

Q1: C — 自适应阈值是光照不均场景的最佳方案（Otsu 仍然是全局的）

Q2: 腐蚀, 膨胀, 去除小白噪点和细连接; 膨胀, 腐蚀, 填补小黑洞和连接断线

Q3: B — 矩形核会在圆形边界产生直角形状的伪影

Q4: 错误 — 腐蚀越多白色区域越小（黑色侵蚀白色），膨胀越多白色区域越大

Q5: 二值化应该放在透视矫正之后。因为倾斜的文档矫正后，文字笔画方向才正，且矫正后光照虽然不均但文档区域是规则的，自适应二值化效果更好。
