### Day 16 Review (2026-07-09) — 形态学操作复盘：腐蚀/膨胀像素级理解 + 开闭运算应用

**完成事项：**
- [x] 像素级可视化：6×6 微图逐格演示 3×3 kernel 滑动机制
- [x] 确认手动计算与 OpenCV 内置 erode/dilate 结果一致
- [x] 4 种场景对比脚本：原图 / 腐蚀 / 膨胀 / 开运算 / 闭运算
- [x] 开闭运算配方逻辑：开=去噪保尺寸，闭=填洞连断线
- [x] 数据验证：开运算只减~800px（噪点），闭运算只增~300px（黑洞+断线）
- [x] `clean_binary()` 练习通过验收（变化 −2.63% < ±5%）
- [x] 删除 `review_dilation_erosion.py`，保留 `review_opening_closing.py`
- [x] 提交分支 `review/morphology-clarification`

**关键理解迭代：**
1. 腐蚀不是"变瘦"而是 AND 逻辑——kernel 下 9 格必须全白才保留
2. 膨胀不是"变胖"而是 OR 逻辑——有白就白，所以噪点也被放大
3. 开闭运算的先后顺序不是随意的一一先开后闭保证外围噪点不污染内部修补
4. `clean_binary()` = open + close 的工业标准流水线

**复盘三问：**
1. 腐蚀和膨胀的本质区别是什么？AND vs OR——腐蚀是黑吃白（收缩），膨胀是白吃黑（扩张）。
2. 什么时候用开运算而不是直接腐蚀？需要去噪但还要保留物体原始大小时——腐蚀+膨胀组合恢复尺寸。
3. 为什么先开再闭？先闭会把外围噪点"封"进物体边缘，开运算虽然能处理但边界会残留凸起伪影。

**复盘三问：**
1. Sobel X 为什么检测垂直边缘？Sobel X 算的是"右边减左边"（沿 X 轴的变化率），亮度沿 X 方向跳变的像素排列恰好是竖线。
2. Canny 滞后连接为什么不用一刀切？双阈值把像素分成三档——强/弱/非——弱边缘必须挂在强边缘上才能活，既干净又连续。
3. Laplacian 真正的用武之地？图像锐化——原图减去 Laplacian 拉大边缘对比度，不是边缘检测。
